== Description ==

WooCommerce GiftProduct provide gift wrap facility to their cart product. Customer can select number of product from their cart product. Also add different delivery address for different product. Customer can also send beautiful mail notification to person, who receive that gift.

Features: 
	- We can select product for gift from list of cart products.
	-  Make One or more group and add different delivery address.
	-  Edit or delete group products and group also.
	-  Gift wrap option for groups.	
	- Email notification to customer when order place.


== Frequently Asked Questions ==

= can we change gift wrap charges ? =
Yes, you can change gift wrap charges from admin Woocommerce -> Settings -> Product tab.
